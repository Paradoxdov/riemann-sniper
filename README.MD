## 📊 Performance
Tested on the **32,132,336,741st** Riemann zero:
* **Prediction Error (L0):** ~0.043
* **Computation Time:** ~7.0 seconds (Pure Python, single thread)
* **Accuracy:** Matches Odlyzko's/Platt's datasets up to the 6th decimal place.

## 🛠 Requirements
* Python 3.7+
* `numpy`
* `scipy`
* `mpmath`
